Available features :
--------------------
Auto-completion : write the beginning of a word, and then the Tab key (as you would do in a terminal)
If a unique command is found with the letters you typed, it will automatically complete.
You can add arguments to the command you typed : for instance, try "cat skills"
You then just have to press Enter, and enjoy the result !
